Sistema Léxico

O sistema léxico são as regras de sintaxe da linguagem que devem ser seguidas para que o navegador possa entender cada instrução declarada no código.

Sistema Léxico do JavaScript

Case-Sensitive
O JavaScript é uma linguagem case-sensitive, ou seja, a forma como são escritas variáveis, funções, comandos e etc são relevantes para o interpretador de códigos do navegador. Então a variável: var nome = 'Wellington Felix'; é diferente da variável var Nome = 'Wellington Felix';
Comentários
Os comentários são linhas que servem de instrução ou anotação para o desenvolvedor, mas são descartadas pelo interpretador de códigos do navegador. No JavaScript podemos usar dois tipos de comentários: // var a = 123; - Comentário de um única linha ou /* */ var a = 132 - Comentário de múltiplas linhas.
Declarações
Uma declaração é uma instrução em código escrita no arquivo JavaScript. Para realizar mais de uma declaração na mesma linha é necessário utilizar o sinal de (;) ao término de cada instrução de código, também é possível dar uma quebra de linha e separar cada declaração em uma linha com ou sem o sinal de (;), mas é uma boa prática utilizar o sinal.
Valores Literais
São valores atribuidos diretamente a variáveis, arrays ou objetos no JavaScript. Exemplos: var nome = 'Wellington Felix' var numero = 123; var objeto { nome: 'Wellington Felix' }; var frutas = ['maça', 'laranja', 'una];.
Caracter de Escape
Podemos escrever strings utilizando uma combinação de aspas duplas com aspas simples ou aspas simples com aspas duplas. O caractere de escape (\) é utilizado quando é necessário abrir, fechar a string e dentro dessa, exibir as aspas duplas ou simples. Exemplo: var frase = "O \"HTML\" é para estrutura" ou var frase = 'O \'HTML\' é para estrutura'
Variáveis
var: Utilizado para receber valores variáveis com escopo global. let: Utilizado para receber valores variáveis com escopo local. const: Utilizado para receber valores imutáveis.
Undefined
Valor atribuído a uma variável ou constante que não possui nenhum valor.
Null
Valor atribuído a uma variável quando se deseja que o seu valor seja nulo.
NaN
Valor atribuído a uma variável quando o resultado de um cálculo não é um número. Uma forma de entender é pensar por exemplo, 2e2 em potência na base 10. 2e2 é o mesmo que 2 x 10² = 200, ou 2e3 é o mesmo que 2 x 10³, ou seja, 2 * 10 * 10 * 10 = 2000. Para o nosso exemplo do tipo de dados infinity, faremos o cálculo 2e308 ou -2e308.
Infinity
São valores muito grande que o JavaScript não consegue representar
Objeto Window
Objeto global (raíz) do JavaScript no browser, dentro dele estão funções como alert e também qualquer variável declarada no escopo do nosso código.
this
Representa o objeto no contexto atual.
