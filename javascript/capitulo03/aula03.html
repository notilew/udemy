Funções

Literais

São funções escrita no formato padrão da linguagem utilizando a palavra reservada function, o nome da função e o corpo da função.

function nomeDaFuncao() {
 return n1 + n2;
}

Funções em JavaScript são cidadãs de primeira classe, isso significa que uma função pode ser tratada como um objeto, ou seja, pode ter propriedades.

nomeDaFuncao.propriedade = 'função que soma dois números.';
 console.log(nomeDaFuncao.propriedade);

Funções em JavaScript podem ser anônimas, ou seja, não precisam possuir um nome, pode-se criar uma variável e atribuir uma função a ela.

var imprimir = function(str) {
 console.log(str);
};
imprimir('testando função anônima');

Funções em JavaScript podem ser passadas como parâmetros.

var imprimir = function(funcao) {
 funcao();
};
imprimir(function() {
 console.log('função passada por parâmetro');
});

Escopo Global e Escopo Local

Em JavaScript qualquer variável criada fora do corpo de uma função possui escopo global, ou seja, essa variável poderá ser acessada através do objeto principal window. As variáveis com escopo global podem ser acessadas dentro de uma função sem a necessidade do comando var, pode-se apenas chamar o nome da variável declarada no escopo gloabal e seu valor poderá ser alterado dentro da função. Já as variáveis criadas dentro do corpo de uma função possui escopo local, nesse caso, essas variáveis só existem dentro da função e enquanto essa estiver sendo executada, ao término da execução as variáveis com escopo local serão deletadas.

Funções AutoExecutáveis - IIFE

Em JavaScript podemos usar o conceito de Funções Auto Executáveis que evita a poluição do objeto window com variáveis globais, utilizando esse conceito em seus scripts todas as variáveis terão escopo local e evitará possivéis conflitos com mesmos nomes de variáveis no escopo global em diferentes scripts. Para isso, é necessário criar uma função anônima envolta por dois parênteses e ao final do parênteses de fechamento, usamos mais dois parênteses indicando que estamos executando essa função quando o script for iniciado.

<script>
 // Função anônima autoexecutável
 (function() {
  var nome = 'wellington felix';
  console.log(nome);  })();
</script>
